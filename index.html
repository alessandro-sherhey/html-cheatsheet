HTML Cheatsheet

This website contains the most important tags you need while writing HTML code. Every tag is inside a card, to possibly make things easier to find and understand.

Basic Structure

<!DOCTYPE html>

This tells the browser that you're writing an HTML 5 code. If you don't put this line of code the browser will guess the version. As you can imagine it often works, but if it guesses the HTML version wrong your website could be a mess!

<html>

This is the first tag you have to write, after the <!DOCTYPE html> tag of course.

<head>

This is the first tag that goes inside the <html> tag. Inside this you'll write all the website metadata.

<body>

This is the most important tag. Here goes what will be shown on the browser!

Headings and Text

<h1>

Headings can have 6 different tags. The first (this one) is the biggest heading while the last is the smallest.

<h2>

The second biggest tag.

<h3>

The third biggest tag.

<h4>

Now to the smaller headings: this is the third smallest tag.

<h5>

The second smallest tag.

<h6>

The smallest tag of all.

<p>

As the name suggests, this tag makes a paragraph in the website.

<a>

Instead this tag is used for links. Instead you'll add a href="" attribute for associating a link with what's inside the <a>. For example if you want to write a paragraph that makes the user go to another website when clicked you need to write:
<a href="yourLink"><p>Click here to go to another website!</p></a>
where yourLink is the link of the external website.

Text Formatting

<strong>

This tag makes text bold. Like this.

<em>

This one makes text italic. "em" stays for emphasis.

<u>

This gives text an underline.

<sub>

This tag is for making text subscript. It can be useful to write scientific formulas. Like H2O.

<sup>

And this is the opposite of subscript. It makes text superscript.

Lists

<ul>

This tag is useful to make lists that don't necessarily need to have every element ordered. "ul" stands for "Unordered List". It just adds a point to every element of the list, like:
• Wake up
• Eat
• Go to sleep

<ol>

This tag, instead of the previous one, makes ordered lists. The lists are by default ordered with numbers, like:
1. Wake up
2. Eat
3. Go to sleep

<li>

This tag is essential to add elements to lists. Every list item needs this tag. For example, the code for the list shown before would be:
<ol>
<li>Wake up</li>
<li>Eat</li>
<li>Go to sleep</li>
</ol>

Tables

<table>

This is where all the content of your table will go. However, this doesn't store the content itself. It serves as a container where all the tags for the structure of the table will go, like rows and data.

<tr>

This tag creates a table row, hence the name.

<td>

This tag goes inside the <td> tag. Here is what will be displayed on the browser! "td" stands for Table Data.

Advanced Structure and Attributes

<div>

This tag is used for grouping different elements together. It is really useful when writing CSS code for the website since you can apply a single property for multiple elements, but that's not the only thing! You can do a lot more things with divs!

id="..."

This is an attribute. It gives an element a specific "name" of your choice, so you can identify it easily when styling with CSS. Note: you can't give the same ID to more elements!

class="..."

This is another attribute that is used for identifying items. But this is different: unlike the id attribute, the same class can be used for more elements!

Semantic HTML

Semantic HTML gives a meaning to the code we write. For example a div, which is a general purpose element, can be an <article>, a <section> an <header>, a <footer>, and much more!

<article>

This tag is commonly used in blogs and posts. It means that the text inside it is not related to other parts of the website.

<nav>

As the name suggests, this is used for navigation bars, like the one on the top of the website!

<header>

This tag is instead used for... headers!

<main>

Inside this tag you'll write all the main content of the website.

<footer>

And inside this one you'll write all the content of the footer.

<header>

This tag is instead used for... headers!

<section>

This tag is used for various sections of content. Imagine if there was a simple website with a whole book in it. Every chapter of the book could go inside the <section> tag!
